# Contributing Guidelines

## Scope
- Detection rules only (Sigma, YARA, Suricata, Snort)
- No PoCs or exploit code
- Defensive use cases only

## Requirements
- Clear description
- MITRE ATT&CK mapping
- False positive notes
- References where applicable

## Pull Requests
- Small, focused changes
- One detection per PR
- Follow repository structure

Low-quality or copy-pasted rules will not be accepted.
